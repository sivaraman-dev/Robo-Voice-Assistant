## 🌟 About The Project

**Robo Voice Assistant** is a real-time voice-controlled robot built for the **Science Day Exhibition** at:

🎓 **Srimath Sivagnana Balaya Swamigal Tamil, Arts and Science College**

This robot listens, understands, processes, and responds to human speech in real time using **local AI models** — even inside a **noisy exhibition environment**.

Unlike cloud assistants, Robo works locally, making it:

- ⚡ Faster  
- 🔒 Private  
- 🧠 Intelligent  
- 🎤 Interactive  

---

## 🧠 The Story Behind Robo

Every robot needs a **body** and a **brain**.

- 🤖 My friends helped design and build the **physical structure** of the robot.

## 🏗️ Tech Stack

### 🎤 Speech Processing
- **Whisper** — Speech Recognition  
- **PyAudio** — Microphone Audio Capture  
- **Porcupine** — Wake Word Detection Engine  

### 🧠 Artificial Intelligence
- **Ollama** — Local AI Runtime  
- **TinyLlama** — Local Large Language Model (LLM)

### 🔊 Voice Output
- **Edge-TTS** — Indian Neural Voice Text-to-Speech

### 💻 Core Development
- **Python**
- **AsyncIO**
- **NumPy**
- **SoundFile**
- **Requests API**

## 🚀 Installation

### 1. Clone Repository
```bash
git clone https://github.com/YOUR_USERNAME/Robo-Voice-Assistant.git
cd Robo-Voice-Assistant
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Install Ollama
Download Ollama from:

👉 [https://ollama.ai](https://ollama.ai)

Run the AI model:
```bash
ollama run tinyllama
```

### 4. Set Porcupine Access Key (Windows)
```cmd
set PORCUPINE_ACCESS_KEY=YOUR_KEY
```

### 5. Run Robo
```bash
your command to run robo.py, e.g.,
python robo.py
```
